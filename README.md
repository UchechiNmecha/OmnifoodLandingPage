## Run Locally

Clone the project

```bash
  git clone https://github.com/UchechiNmecha/OmnifoodLandingPage.git
```

Go to the project directory

```bash
  cd OmnifoodLandingPage
```

Live Server Installation

```bash
  npm install -g live-server
```

Start the server

```bash
  live-server
```
